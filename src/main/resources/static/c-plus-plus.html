C++

The C language is a programming language created in the early 1970s by Dennis Ritchie for programming operating systems and other low-level tasks. The C++ language is a multi-paradigm programming language, among other things also object-oriented. It was created in the 1980s by Bjarne Stroustrup. In addition to C as its basis, the C++ language has been influenced primarily by Simula, from which it took object-oriented properties, as well as languages such as Algol, Ada, ML and Clu.

Language basics

Syntax - Control statements - Operators - Data types - Functions - Namespaces - Arrays - Character strings

About C++

C++ is a powerful programming language that supports various programming styles such as procedural, object-oriented, and generic programming. It is widely used for system/software development, game development, and in performance-critical applications. C++ provides extensive control over system resources and memory, making it a preferred choice for high-performance applications.

Prerequisite to Learn C++

Before diving into C++, it is beneficial to have a basic understanding of programming concepts, such as variables, loops, and conditionals. Familiarity with the C programming language can be an advantage but is not mandatory. Problem-solving skills, logical thinking, and a willingness to tackle complex concepts are essential for mastering C++.

C++ Features

  • Multi-paradigm support (procedural, object-oriented, generic)
  • Performance and efficiency
  • Low-level memory manipulation
  • Extensive Standard Library
  • Rich set of libraries and frameworks
  • Strong type checking
  • Compatibility with C

Object-oriented programming

Classes - Access control - Constructors - Destructors - Inheritance - Virtual functions - Abstract classes - Overloading operators

Object-oriented programming (OOP) in C++ is a paradigm that uses "objects" to design applications and programs. Key OOP concepts in C++ include classes, objects, inheritance, polymorphism, encapsulation, and abstraction. These principles help in creating modular, reusable, and maintainable code, making it easier to manage and scale complex software systems.

Advanced issues

Exception handling - Function templates - Class templates

C++ provides advanced features such as exception handling, templates, and the Standard Template Library (STL). Exception handling helps manage errors and exceptions in a controlled manner. Templates enable generic programming by allowing functions and classes to operate with generic types. These advanced features contribute to the robustness and flexibility of C++ programming.

Standard library

Stdio.h (cstdio) - Stdlib.h (cstdlib) - String.h (cstring) - Time.h (ctime) - Math.h (cmath) - Iostream - Standard Templates Library

The C++ Standard Library provides a set of common classes and functions for tasks such as input/output operations, string manipulation, and mathematical computations. The Standard Template Library (STL) is a powerful set of C++ template classes that provide general-purpose classes and functions with templates that implement many popular and commonly used algorithms and data structures like vectors, lists, and stacks.
