Data Types in C#

Data types in C# define the type of data that a variable can hold. Here are some common data types in C#:

  • int: Used to store integer values.
  • double: Used to store floating-point values with double precision.
  • bool: Used to store Boolean values (true or false).
  • char: Used to store single characters.
  • string: Used to store sequences of characters.
  • decimal: Used to store decimal values with higher precision.
  • float: Used to store floating-point values with single precision.

Here's an example demonstrating the usage of some data types in C#: 

        
using System;

class Program
{
    static void Main()
    {
        int age = 30;
        double height = 5.9;
        bool isStudent = true;
        char grade = 'A';
        string name = "John Doe";
        decimal salary = 50000.50m;
        float weight = 70.5f;

        Console.WriteLine("Name: " + name);
        Console.WriteLine("Age: " + age);
        Console.WriteLine("Height: " + height);
        Console.WriteLine("Is Student: " + isStudent);
        Console.WriteLine("Grade: " + grade);
        Console.WriteLine("Salary: " + salary);
        Console.WriteLine("Weight: " + weight);
    }
}
