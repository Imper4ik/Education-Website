Standard Template Library (STL) in C#

In C++, the Standard Template Library (STL) provides a collection of generic classes and functions, such as vectors, lists, and algorithms, which help in managing data and performing operations on them. In C#, similar functionality is provided by the System.Collections.Generic namespace.

Here's an example demonstrating the usage of some generic collections in C#:

        
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // List example
        List numbers = new List { 1, 2, 3, 4, 5 };
        numbers.Add(6);
        numbers.Remove(2);
        Console.WriteLine("List elements:");
        foreach (int number in numbers)
        {
            Console.WriteLine(number);
        }

        // Dictionary example
        Dictionary ages = new Dictionary();
        ages["Alice"] = 30;
        ages["Bob"] = 25;
        Console.WriteLine("\nDictionary elements:");
        foreach (KeyValuePair kvp in ages)
        {
            Console.WriteLine("Key = {0}, Value = {1}", kvp.Key, kvp.Value);
        }
    }
}

In this example, we use a List<T> to store a collection of integers and a Dictionary<TKey, TValue> to store key-value pairs. These collections provide powerful methods and properties to manage data efficiently.

C# also provides other generic collections, such as Queue<T>, Stack<T>, and HashSet<T>, which offer different functionalities and performance characteristics.
