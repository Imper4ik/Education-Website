Math.h (cmath) in C#

In C#, the functionality provided by `math.h` (also known as `cmath` in C++) is available through the `System.Math` class. This class provides methods for performing mathematical operations such as trigonometric, logarithmic, and exponential calculations.

Here's an example demonstrating some common mathematical operations in C#:

        
using System;

class Program
{
    static void Main()
    {
        // Trigonometric functions
        double angle = Math.PI / 4; // 45 degrees
        double sine = Math.Sin(angle);
        double cosine = Math.Cos(angle);
        double tangent = Math.Tan(angle);

        Console.WriteLine("Sine of 45 degrees: " + sine);
        Console.WriteLine("Cosine of 45 degrees: " + cosine);
        Console.WriteLine("Tangent of 45 degrees: " + tangent);

        // Logarithmic functions
        double number = 10;
        double log10 = Math.Log10(number);
        double naturalLog = Math.Log(number);

        Console.WriteLine("Log10 of 10: " + log10);
        Console.WriteLine("Natural log of 10: " + naturalLog);

        // Exponential functions
        double exponent = 2;
        double exp = Math.Exp(exponent);
        double power = Math.Pow(number, exponent);

        Console.WriteLine("Exp(2): " + exp);
        Console.WriteLine("10^2: " + power);

        // Other mathematical functions
        double sqrt = Math.Sqrt(number);
        double abs = Math.Abs(-number);
        double ceiling = Math.Ceiling(4.3);
        double floor = Math.Floor(4.7);

        Console.WriteLine("Square root of 10: " + sqrt);
        Console.WriteLine("Absolute value of -10: " + abs);
        Console.WriteLine("Ceiling of 4.3: " + ceiling);
        Console.WriteLine("Floor of 4.7: " + floor);
    }
}

In this example, we demonstrate trigonometric functions, logarithmic functions, exponential functions, and other common mathematical operations using the `System.Math` class. C# provides a comprehensive set of features for performing mathematical calculations.
