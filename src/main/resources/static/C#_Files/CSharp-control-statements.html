Control Statements in C#

C# provides various control statements that allow you to control the flow of execution in your program. These include:

  • if-else: Executes a block of code if a specified condition is true, otherwise executes another block of code.
  • while: Repeats a block of code as long as a specified condition is true.
  • do-while: Similar to the while loop, but the code block is executed at least once before the condition is tested.
  • for: Executes a block of code a specified number of times.
  • break: Terminates the loop or switch statement and transfers control to the statement immediately following the loop or switch.
  • continue: Skips the current iteration of a loop and continues with the next iteration.

Example:

        
using System;

class Program
{
    static void Main()
    {
        // if-else statement
        int x = 10;
        if (x > 5)
        {
            Console.WriteLine("x is greater than 5");
        }
        else
        {
            Console.WriteLine("x is not greater than 5");
        }

        // while loop
        int y = 0;
        while (y < 5)
        {
            Console.WriteLine(y);
            y++;
        }

        // for loop
        for (int i = 0; i < 5; i++)
        {
            Console.WriteLine(i);
        }
    }
}
