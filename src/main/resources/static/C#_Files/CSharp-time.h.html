Time.h (ctime) in C#

In C#, the functionality provided by `time.h` (also known as `ctime` in C++) is available through the `System.DateTime` and `System.TimeSpan` classes. These classes provide methods and properties for handling dates, times, and time intervals.

Here's an example demonstrating some common date and time operations in C#:

        
using System;

class Program
{
    static void Main()
    {
        // Current date and time
        DateTime now = DateTime.Now;
        Console.WriteLine("Current date and time: " + now);

        // Specific date and time
        DateTime specificDate = new DateTime(2023, 6, 13, 12, 0, 0);
        Console.WriteLine("Specific date and time: " + specificDate);

        // Add time
        DateTime futureDate = now.AddDays(10);
        Console.WriteLine("Date 10 days from now: " + futureDate);

        // Time difference
        TimeSpan difference = futureDate - now;
        Console.WriteLine("Time difference: " + difference.Days + " days");

        // Formatting date and time
        string formattedDate = now.ToString("yyyy-MM-dd HH:mm:ss");
        Console.WriteLine("Formatted date and time: " + formattedDate);
    }
}

In this example, we demonstrate getting the current date and time, creating a specific date and time, adding time, calculating the difference between dates, and formatting dates using the `System.DateTime` and `System.TimeSpan` classes. C# provides a comprehensive set of features for handling and manipulating dates and times.
