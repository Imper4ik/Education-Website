String.h (cstring) in C#

In C#, the functionality provided by `string.h` (also known as `cstring` in C++) is available through the `System.String` class and various methods provided by the .NET framework. These include methods for string manipulation, comparison, searching, and more.

Here's an example demonstrating some common string operations in C#:

        
using System;

class Program
{
    static void Main()
    {
        string str1 = "Hello";
        string str2 = "World";

        // Concatenation
        string result = string.Concat(str1, " ", str2);
        Console.WriteLine("Concatenation: " + result);

        // Comparison
        int comparison = string.Compare(str1, str2);
        Console.WriteLine("Comparison: " + comparison);

        // Searching
        bool contains = str1.Contains("ll");
        Console.WriteLine("Contains 'll': " + contains);

        // Substring
        string substring = str1.Substring(1, 3);
        Console.WriteLine("Substring: " + substring);

        // Length
        int length = str1.Length;
        Console.WriteLine("Length: " + length);
    }
}

In this example, we demonstrate string concatenation, comparison, searching, extracting a substring, and finding the length of a string using the `System.String` class methods. C# provides a rich set of features for handling and manipulating strings.
