C++ I/O with iostream

Introduction

The `` library in C++ provides classes and objects to perform input and output operations. It includes the `cin` for input and `cout` for output, among others.

Basic Usage

Here's a simple example demonstrating the use of `` for input and output:

#include <iostream>

int main() {
    // Output to console
    std::cout << "Hello, World!" << std::endl;

    // Input from console
    int num;
    std::cout << "Enter a number: ";
    std::cin >> num;
    std::cout << "You entered: " << num << std::endl;

    return 0;
}

This example outputs a greeting message and prompts the user to enter a number, then displays the entered number.

Formatted Output

You can format output using stream manipulators:

#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.14159265359;
    std::cout << "Pi value with precision: " << std::fixed << std::setprecision(4) << pi << std::endl;

    return 0;
}

This code snippet formats the output of the value of Pi with a specific precision.

Summary

The `` library in C++ provides a convenient way to perform input and output operations using `cin` for input and `cout` for output. It supports formatted output using stream manipulators for precision and other formatting options.
