C++ Inheritance

Introduction

Inheritance in C++ allows one class (derived class) to inherit properties and behaviors from another class (base class). This enables code reusability and the creation of hierarchical relationships between classes.

Syntax of Inheritance

To derive a class in C++, use the following syntax:

class DerivedClass : accessSpecifier BaseClass {
    // Derived class members
};

Here, DerivedClass inherits from BaseClass with the specified accessSpecifier (public, protected, or private).

Types of Inheritance

C++ supports several types of inheritance:

  • Single inheritance: One derived class inherits from one base class.
  • Multiple inheritance: One derived class inherits from multiple base classes.
  • Multi-level inheritance: Deriving a class from another derived class.
  • Hierarchical inheritance: Multiple derived classes inherit from a single base class.
  • Hybrid inheritance: Combination of two or more types of inheritance.

Example of Inheritance

Below is an example demonstrating single inheritance:

class Animal {
public:
    void eat() {
        cout << "Animal eats" << endl;
    }
};

class Dog : public Animal {
public:
    void bark() {
        cout << "Dog barks" << endl;
    }
};

int main() {
    Dog dog;
    dog.eat();  // Accessing base class method
    dog.bark(); // Accessing derived class method
    return 0;
}

Benefits of Inheritance

Using inheritance offers several advantages:

  • Code reuse: Avoids redundant code by inheriting existing functionalities.
  • Modularity: Classes can be easily extended and modified.
  • Hierarchical structure: Facilitates modeling real-world relationships.
  • Polymorphism: Allows objects to be treated uniformly through base class pointers or references.

Summary

Inheritance is a powerful feature of C++ that supports the creation of hierarchies and facilitates code reuse. Understanding its syntax and types helps in designing efficient and maintainable object-oriented programs.
