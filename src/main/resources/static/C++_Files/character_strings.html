C++ Character Strings

Introduction

Character strings are arrays of characters in C++. They are used to store sequences of characters, such as words or sentences. C++ provides several ways to handle character strings, including C-style strings and the C++ string class.

C-Style Strings

C-style strings are arrays of characters terminated by a null character ('\0'). They can be declared and initialized as follows:

char str1[] = "Hello, World!";  // String literal
char str2[13] = "Hello, World!";  // Specifying the size
char str3[] = {'H', 'e', 'l', 'l', 'o', ',', ' ', 'W', 'o', 'r', 'l', 'd', '!', '\0'};  // Character array

String Input and Output

To read a C-style string from the user, you can use cin. To output a string, you can use cout:

char name[50];
cout << "Enter your name: ";
cin >> name;
cout << "Hello, " << name << "!";

String Manipulation Functions

C++ provides a number of functions for manipulating C-style strings, which are available in the <cstring> header. Some commonly used functions include:

  • strlen(str) - Returns the length of the string str
  • strcpy(dest, src) - Copies the string src to dest
  • strcat(dest, src) - Appends the string src to dest
  • strcmp(str1, str2) - Compares the strings str1 and str2
char str1[20] = "Hello";
char str2[20] = "World";
char str3[40];

strcpy(str3, str1);  // Copy str1 into str3
strcat(str3, " ");   // Concatenate a space to str3
strcat(str3, str2);  // Concatenate str2 to str3

cout << "Combined String: " << str3;  // Output: "Hello World"

C++ string Class

The C++ Standard Library provides the string class to handle strings more easily and safely than C-style strings. To use it, include the <string> header:

#include <string>
using namespace std;

string str1 = "Hello";
string str2 = "World";
string str3 = str1 + " " + str2;

cout << "Combined String: " << str3;  // Output: "Hello World"

Common string Class Methods

The string class provides many useful methods for string manipulation:

  • length() - Returns the length of the string
  • append(str) - Appends str to the string
  • substr(pos, len) - Returns a substring starting at pos with length len
  • find(str) - Returns the position of the first occurrence of str in the string
  • replace(pos, len, str) - Replaces a portion of the string starting at pos with length len with str
string str = "Hello, World!";
cout << "Length: " << str.length();  // Output: 13
cout << "Substring: " << str.substr(7, 5);  // Output: "World"
str.replace(7, 5, "C++");
cout << "Modified String: " << str;  // Output: "Hello, C++!"
