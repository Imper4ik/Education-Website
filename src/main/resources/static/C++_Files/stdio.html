C++ Standard Library: cstdio

Introduction

The `` library in C++ (or `stdio.h` in C) provides functions for input and output operations using the standard streams. It includes functions for reading and writing files, formatting output, and handling errors.

Basic Input and Output

Here are some basic input and output functions provided by ``:

#include <iostream>
#include <cstdio>

int main() {
    // Basic output using printf
    std::printf("Hello, world! This is printed using printf.\n");

    // Basic input using scanf
    int number;
    std::printf("Enter a number: ");
    std::scanf("%d", &number);
    std::printf("You entered: %d\n", number);

    return 0;
}

This example demonstrates the use of `std::printf` for output and `std::scanf` for input operations.

File Operations

Functions for file input and output:

#include <iostream>
#include <cstdio>

int main() {
    // File operations example
    FILE* file = std::fopen("example.txt", "w");
    if (file != nullptr) {
        std::fprintf(file, "This is written to example.txt using fprintf.\n");
        std::fclose(file);
        std::printf("File written successfully.\n");
    } else {
        std::printf("Failed to open file.\n");
    }

    return 0;
}

This code snippet demonstrates how to use `std::fopen`, `std::fprintf`, and `std::fclose` for file operations.

Error Handling

Functions for error handling:

#include <iostream>
#include <cstdio>

int main() {
    // Error handling example
    int result = std::remove("nonexistent_file.txt");
    if (result == 0) {
        std::printf("File deleted successfully.\n");
    } else {
        std::perror("Error deleting file");
    }

    return 0;
}

This snippet uses `std::remove` for deleting a file and demonstrates error handling with `std::perror`.
