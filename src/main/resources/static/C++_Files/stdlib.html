C++ Standard Library: cstdlib

Introduction

The `` library in C++ (or `stdlib.h` in C) provides general utilities, including functions for memory allocation, random number generation, and string manipulation.

Memory Allocation

Here's an example using memory allocation functions:

#include <iostream>
#include <cstdlib>

int main() {
    // Allocate memory for an array of integers
    int* array = (int*)std::malloc(5 * sizeof(int));
    if (array != nullptr) {
        for (int i = 0; i < 5; ++i) {
            array[i] = i + 1;
        }
        std::cout << "Array elements: ";
        for (int i = 0; i < 5; ++i) {
            std::cout << array[i] << " ";
        }
        std::cout << std::endl;
        std::free(array);
    }

    return 0;
}

This example demonstrates the use of `std::malloc` for dynamic memory allocation and `std::free` for freeing allocated memory.

String Manipulation

Functions for string manipulation:

#include <iostream>
#include <cstdlib>
#include <cstring>

int main() {
    // String manipulation example
    char str[] = "Hello, world!";
    std::cout << "Length of string: " << std::strlen(str) << std::endl;

    return 0;
}

This code snippet shows how to use `std::strlen` to get the length of a C-style string.

Random Number Generation

Functions for random number generation:

#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {
    // Random number generation
    std::srand(std::time(nullptr)); // seed random number generator
    int randomNumber = std::rand() % 100 + 1; // generate random number between 1 and 100
    std::cout << "Random number: " << randomNumber << std::endl;

    return 0;
}

This snippet seeds the random number generator with `std::srand` and generates a random number using `std::rand`.
