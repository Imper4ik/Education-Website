C++ Data Types

Introduction

Data types in C++ specify the type of data that a variable can hold. C++ provides several built-in data types, as well as the ability to define your own types using structures, classes, and enumerations.

Basic Data Types

Data Type Description Size
int Integer type Usually 4 bytes
float Floating point type Usually 4 bytes
double Double precision floating point type Usually 8 bytes
char Character type 1 byte
bool Boolean type 1 byte

Derived Data Types

Derived data types include arrays, pointers, references, and function types.

  • int arr[10]; - Array of integers
  • int* ptr; - Pointer to an integer
  • int& ref = var; - Reference to an integer
  • int func(); - Function returning an integer

User-defined Data Types

User-defined data types include structures, unions, classes, and enumerations.

  • struct - Defines a structure
  • union - Defines a union
  • class - Defines a class
  • enum - Defines an enumeration
struct Person {
    std::string name;
    int age;
};

union Data {
    int intValue;
    float floatValue;
};

class Animal {
public:
    std::string species;
    void makeSound() {
        std::cout << "Sound!" << std::endl;
    }
};

enum Color {
    RED,
    GREEN,
    BLUE
};
