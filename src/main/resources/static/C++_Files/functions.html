C++ Functions

Introduction

Functions are a fundamental building block in C++ programming. They allow you to encapsulate code into reusable blocks that can be called multiple times throughout a program. Functions help make programs more modular, readable, and maintainable.

Defining a Function

A function definition in C++ consists of a return type, function name, parameters, and a body. Here's the basic syntax:

return_type function_name(parameters) {
    // function body
}

Example:

int add(int a, int b) {
    return a + b;
}

Calling a Function

Once a function is defined, it can be called from anywhere in the program. When calling a function, you provide arguments for its parameters:

int result = add(5, 3);  // result will be 8

Function Parameters

Functions can take zero or more parameters. Parameters act as placeholders for the values passed into the function when it is called. Parameters are specified within the parentheses in the function definition.

void greet(std::string name) {
    std::cout << "Hello, " << name << "!" << std::endl;
}

Example of calling the function:

greet("Alice");  // Output: Hello, Alice!

Return Values

Functions can return values of any data type. The return type is specified before the function name. Use the return statement to return a value from the function.

double multiply(double x, double y) {
    return x * y;
}

Example of calling the function:

double product = multiply(2.5, 4.0);  // product will be 10.0

Function Overloading

C++ allows you to define multiple functions with the same name but different parameter lists. This is called function overloading. The correct function is chosen based on the arguments provided during the function call.

int multiply(int a, int b) {
    return a * b;
}

double multiply(double x, double y) {
    return x * y;
}

Example of calling the overloaded functions:

int product1 = multiply(3, 4);        // Calls int version, product1 will be 12
double product2 = multiply(2.5, 4.0);  // Calls double version, product2 will be 10.0
