C++ Operator Overloading

Introduction

Operator overloading in C++ allows you to redefine the way operators work for user-defined types (classes and structs). It enables you to extend the functionality of operators to work with objects.

Overloading Operators

To overload an operator, you define a function with the keyword operator followed by the operator symbol or keyword:

class Complex {
private:
    double real;
    double imaginary;
public:
    Complex(double r, double i) : real(r), imaginary(i) {}

    // Overloading the + operator
    Complex operator+(const Complex& other) const {
        return Complex(real + other.real, imaginary + other.imaginary);
    }

    // Overloading the * operator
    Complex operator*(const Complex& other) const {
        return Complex(real * other.real - imaginary * other.imaginary,
                       real * other.imaginary + imaginary * other.real);
    }

    // Overloading the << operator (as a friend function)
    friend ostream& operator<<(ostream& os, const Complex& c) {
        os << c.real << " + " << c.imaginary << "i";
        return os;
    }
};

int main() {
    Complex a(1.0, 2.0);
    Complex b(3.0, 4.0);

    Complex c = a + b; // Calls operator+
    Complex d = a * b; // Calls operator*

    cout << "a + b = " << c << endl;
    cout << "a * b = " << d << endl;

    return 0;
}

In this example, Complex class overloads the addition (+), multiplication (*), and insertion (<<) operators. Overloaded operators can be member functions or non-member functions (often friend functions).

Commonly Overloaded Operators

Some commonly overloaded operators include:

  • +, -, *, / for arithmetic operations
  • +=, -=, *=, /= for compound assignment
  • ++, -- for prefix and postfix increment/decrement
  • ==, !=, <, >, <=, >= for comparison
  • [], (), -> for subscripting, function call, and member access
  • =, <<, >> for assignment, output, and input

Considerations

Operator overloading should be used judiciously to maintain clarity and consistency in code. Overloaded operators should mimic the behavior expected from built-in types to avoid confusion.

Summary

Operator overloading in C++ allows you to redefine the behavior of operators for user-defined types, providing flexibility and enabling intuitive use of objects in expressions. Understanding how to correctly overload operators is important for creating expressive and maintainable code.
