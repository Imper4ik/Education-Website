C++ Classes

Introduction

Classes are user-defined types in C++ that represent real-world entities. They allow you to bundle data and functions together to form objects. Classes form the foundation of Object-Oriented Programming (OOP) in C++.

Defining a Class

A class definition starts with the class keyword, followed by the class name and a pair of curly braces containing its members:

class MyClass {
public:
    int myNumber;
    void myFunction() {
        cout << "Hello, World!";
    }
};

Creating Objects

Once a class is defined, you can create objects (instances of the class) using the class name:

MyClass myObj;  // Create an object of MyClass
myObj.myNumber = 5;  // Access the data member
myObj.myFunction();  // Call the member function

Access Specifiers

Access specifiers define the access level of class members. The three access specifiers are:

  • public: Members are accessible from outside the class.
  • private: Members are accessible only within the class.
  • protected: Members are accessible within the class and by derived classes.
class MyClass {
public:
    int publicNumber;
private:
    int privateNumber;
};

Constructors and Destructors

Constructors are special member functions that are called when an object is created. Destructors are called when an object is destroyed.

class MyClass {
public:
    // Constructor
    MyClass() {
        cout << "Object is created";
    }
    // Destructor
    ~MyClass() {
        cout << "Object is destroyed";
    }
};

Member Functions

Member functions can be defined inside or outside the class definition. When defined outside, use the scope resolution operator ::.

class MyClass {
public:
    void myFunction();  // Declaration
};

// Definition outside the class
void MyClass::myFunction() {
    cout << "Hello, World!";
}

Encapsulation

Encapsulation is the concept of wrapping data and functions that operate on the data into a single unit. It helps in data hiding and abstraction.

class MyClass {
private:
    int myNumber;
public:
    void setNumber(int num) {
        myNumber = num;
    }
    int getNumber() {
        return myNumber;
    }
};
