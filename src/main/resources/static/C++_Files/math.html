C++ Math with cmath

Introduction

The `` library in C++ provides a set of common mathematical functions and constants. It includes functions for basic arithmetic, trigonometry, logarithms, exponentiation, and more.

Basic Functions

Here are some basic mathematical functions provided by ``:

#include <iostream>
#include <cmath>

int main() {
    // Square root
    double number = 16.0;
    double squareRoot = std::sqrt(number);
    std::cout << "Square root of " << number << " is " << squareRoot << std::endl;

    // Trigonometric functions
    double angle = 45.0;
    double sineValue = std::sin(angle * std::acos(-1.0) / 180.0);
    std::cout << "Sine of " << angle << " degrees is " << sineValue << std::endl;

    return 0;
}

This example demonstrates the use of `std::sqrt` for computing square roots and `std::sin` for computing sine values.

Constants

The `` library also provides mathematical constants:

#include <cmath>

int main() {
    // Pi constant
    double piValue = std::acos(-1.0);
    std::cout << "The value of Pi is approximately " << piValue << std::endl;

    // Euler's number (e)
    double eValue = std::exp(1.0);
    std::cout << "The value of e (Euler's number) is approximately " << eValue << std::endl;

    return 0;
}

This code snippet shows how to retrieve the values of Pi and Euler's number using ``.

The `` library in C++ provides essential mathematical functions and constants for performing common calculations. It enhances productivity by offering ready-to-use implementations of mathematical operations.
