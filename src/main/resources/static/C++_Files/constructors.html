C++ Constructors

Introduction

Constructors in C++ are special member functions that are executed whenever an object of a class is created. They are used to initialize objects.

Default Constructor

A default constructor is a constructor that takes no parameters. If no constructor is defined, the compiler generates a default constructor.

class MyClass {
public:
    MyClass() {
        cout << "Default constructor called!";
    }
};

int main() {
    MyClass obj; // Default constructor is called
    return 0;
}

Parameterized Constructor

A parameterized constructor is a constructor that takes arguments. It allows initializing an object with specific values.

class MyClass {
private:
    int value;
public:
    MyClass(int v) {
        value = v;
    }
    void display() {
        cout << "Value: " << value;
    }
};

int main() {
    MyClass obj(10); // Parameterized constructor is called
    obj.display();
    return 0;
}

Copy Constructor

A copy constructor is a constructor that initializes an object using another object of the same class. It takes a reference to an object of the same class as a parameter.

class MyClass {
private:
    int value;
public:
    MyClass(int v) {
        value = v;
    }
    MyClass(const MyClass &obj) {
        value = obj.value;
    }
    void display() {
        cout << "Value: " << value;
    }
};

int main() {
    MyClass obj1(10); // Parameterized constructor is called
    MyClass obj2 = obj1; // Copy constructor is called
    obj2.display();
    return 0;
}

Destructor

A destructor is a special member function that is executed when an object of a class is destroyed. It is used to release resources allocated by the object.

class MyClass {
public:
    ~MyClass() {
        cout << "Destructor called!";
    }
};

int main() {
    MyClass obj; // Destructor will be called when obj goes out of scope
    return 0;
}

Summary

Constructors are essential for initializing objects in C++. They provide a mechanism to set initial values and allocate resources when an object is created. Destructors complement constructors by cleaning up resources when an object is destroyed.
