Basic C++ Syntax

Introduction

C++ is a powerful general-purpose programming language. It is used for creating high-performance applications. Here are some of the basic syntax elements of C++.

Hello, World!

The following example shows a simple C++ program that prints "Hello, World!" to the console.

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

Variables

Variables are used to store data. In C++, you must declare a variable before you use it. Here is an example of variable declaration and initialization:

int myNumber = 5;
double myFloat = 5.99;
char myLetter = 'D';

Data Types

C++ supports several different data types, including:

  • int: Integer type
  • double: Floating point type
  • char: Character type
  • bool: Boolean type
  • string: String type (provided by the standard library)

Control Structures

Control structures in C++ include:

If-Else

int x = 10;
if (x > 0) {
    std::cout << "x is positive" << std::endl;
} else {
    std::cout << "x is not positive" << std::endl;
}

For Loop

for (int i = 0; i < 5; i++) {
    std::cout << i << std::endl;
}

While Loop

int i = 0;
while (i < 5) {
    std::cout << i << std::endl;
    i++;
}

Functions

Functions are blocks of code that perform a specific task. Here is an example of a function in C++:

void myFunction() {
    std::cout << "Hello from myFunction!" << std::endl;
}

int main() {
    myFunction();
    return 0;
}
