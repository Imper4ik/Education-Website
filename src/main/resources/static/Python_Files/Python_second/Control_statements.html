Python Control Statements

Introduction

Control statements in Python allow you to control the flow of execution based on conditions or loops. They are essential for making decisions and iterating over data.

If Statement

The `if` statement executes a block of code only if a specified condition is true:

num = 10
if num > 0:
    print("Number is positive")

Python uses indentation (whitespace at the beginning of a line) to indicate code blocks.

Else Statement

The `else` statement follows an `if` statement and executes a block of code if the condition is false:

num = -5
if num > 0:
    print("Number is positive")
else:
    print("Number is non-positive")

Elif Statement

The `elif` statement allows you to check multiple conditions sequentially:

num = 0
if num > 0:
    print("Number is positive")
elif num < 0:
    print("Number is negative")
else:
    print("Number is zero")

For Loop

The `for` loop is used for iterating over a sequence (such as a list, tuple, string, or range):

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

While Loop

The `while` loop executes a block of code as long as a specified condition is true:

num = 1
while num <= 5:
    print(num)
    num += 1

Break and Continue

The `break` statement terminates the loop immediately, and `continue` skips the current iteration and moves to the next:

for num in range(1, 6):
    if num == 3:
        continue
    print(num)
    if num == 4:
        break

Summary

Control statements such as `if`, `else`, loops (`for` and `while`), `break`, and `continue` are fundamental in Python for decision-making and iterative tasks. Mastering these statements allows you to write efficient and structured Python programs.
