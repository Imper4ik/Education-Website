Python Character Strings

Introduction

Character strings (or strings) in Python are sequences of characters enclosed in single quotes (' ') or double quotes (" "). They are used to store and manipulate textual data.

Creating Strings

Strings can be created using single quotes or double quotes:

single_quoted_string = 'Hello, Python!'
double_quoted_string = "Python is awesome!"

Accessing Characters

You can access characters in a string using indexing:

my_string = "Hello"
first_char = my_string[0]  # 'H'
last_char = my_string[-1]  # 'o'

String Operations

Python provides several operations for manipulating strings:

  • len() - Returns the length of the string.
  • concatenation (+) - Combines two strings.
  • repetition (*) - Repeats a string a specified number of times.
  • in - Checks if a substring exists in the string.
message = "Hello"
length = len(message)  # 5
combined = message + ", World!"  # 'Hello, World!'
repeated = message * 3  # 'HelloHelloHello'
contains_o = 'o' in message  # True

String Methods

Python strings have built-in methods for various operations:

my_string = " Hello, Python! "
lowercase = my_string.lower()  # ' hello, python! '
uppercase = my_string.upper()  # ' HELLO, PYTHON! '
stripped = my_string.strip()  # 'Hello, Python!'
split_words = my_string.split(',')  # [' Hello', ' Python! ']

Formatting Strings

Strings can be formatted using:

  • format() method
  • F-strings (formatted string literals)
name = "Alice"
age = 30
formatted_string = "Name: {}, Age: {}".format(name, age)  # 'Name: Alice, Age: 30'
f_string = f"Name: {name}, Age: {age}"  # 'Name: Alice, Age: 30'

Summary

Strings in Python are versatile and powerful for handling textual data. Understanding how to create, manipulate, and format strings is essential for effective programming in Python.
