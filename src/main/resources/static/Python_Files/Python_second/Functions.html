Python Functions

Introduction

Functions in Python are blocks of reusable code that perform a specific task. They help in organizing code into manageable pieces and promote code reusability.

Defining a Function

A function is defined using the def keyword, followed by the function name and parameters (if any). It may return a value using the return statement.

def greet(name):
    return f"Hello, {name}!"

message = greet("Alice")  # 'Hello, Alice!'

Function Parameters

Functions can have parameters (inputs) that allow the function to accept data or arguments. Parameters can have default values.

def add_numbers(x, y=0):
    return x + y

result1 = add_numbers(5, 3)  # 8
result2 = add_numbers(5)     # 5 (default y=0 is used)

Returning Values

Functions can return values using the return statement. They can return multiple values as a tuple.

def calculate(x, y):
    sum = x + y
    difference = x - y
    return sum, difference

result = calculate(10, 5)
# result is a tuple: (15, 5)

Function Docstring

It's good practice to include a docstring at the beginning of a function to describe its purpose, parameters, and return value.

def square(x):
    """
    Returns the square of a number.

    Parameters:
    x (int): The number to be squared.

    Returns:
    int: The square of x.
    """
    return x ** 2

Anonymous Functions (Lambda Functions)

Python supports anonymous functions using the lambda keyword. They are typically used for short functions.

double = lambda x: x * 2
result = double(5)  # 10

Summary

Functions are essential for structuring Python code into reusable blocks. They enhance code readability, maintainability, and promote code reuse, contributing to efficient and organized programming.
