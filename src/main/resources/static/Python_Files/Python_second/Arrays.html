Python Arrays

Introduction

Arrays in Python are data structures used to store multiple elements of the same type. Unlike some other programming languages, Python does not have a built-in array type, but it provides several alternatives, such as lists and NumPy arrays, which serve similar purposes.

Lists

Lists are the most common way to work with arrays-like structures in Python:

my_list = [1, 2, 3, 4, 5]
print("Elements of the list:")
for element in my_list:
    print(element)

Lists can store elements of different data types and are versatile for many programming tasks.

NumPy Arrays

NumPy is a powerful library for numerical computing in Python, providing an array object:

import numpy as np

my_array = np.array([1, 2, 3, 4, 5])
print("Elements of the NumPy array:")
for element in my_array:
    print(element)

NumPy arrays are efficient for numerical operations and allow multidimensional arrays and advanced operations.

Accessing Elements

Accessing elements in arrays:

my_list = [10, 20, 30, 40, 50]
print("First element:", my_list[0])
print("Last element:", my_list[-1])

Arrays in Python are accessed using indexing, starting from 0, and negative indices for counting from the end.

Manipulating Arrays

Manipulating arrays in Python:

import numpy as np

# Create a NumPy array
arr = np.array([1, 2, 3, 4, 5])

# Append elements
arr = np.append(arr, 6)
print("Appended array:", arr)

# Sorting elements
sorted_arr = np.sort(arr)
print("Sorted array:", sorted_arr)

Arrays can be modified by adding elements, sorting, and performing various operations based on the array type.

Summary

Arrays in Python, primarily represented using lists and NumPy arrays, are fundamental for storing and manipulating collections of elements. Understanding their usage and operations is crucial for effective Python programming, especially in data manipulation and scientific computing.
