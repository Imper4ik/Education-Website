Python Operators

Introduction

Operators in Python are symbols or special functions that perform operations on operands (variables and values). They are classified into different types such as arithmetic, comparison, logical, assignment, and more.

Arithmetic Operators

Arithmetic operators are used for mathematical calculations:

+, -, *, /, %, **

Comparison Operators

Comparison operators are used to compare values:

==, !=, >, <, >=, <=

Logical Operators

Logical operators are used for combining conditional statements:

and, or, not

Assignment Operators

Assignment operators are used to assign values to variables:

=, +=, -=, *=, /=, %=, **=

Bitwise Operators

Bitwise operators operate on bits and perform bit-by-bit operations:

&, |, ^, ~, <<, >>

Identity Operators

Identity operators compare the memory location of two objects:

is, is not

Membership Operators

Membership operators check for membership in sequences (lists, tuples, strings):

in, not in

Operator Precedence

Operators in Python have precedence levels that dictate the order of operations:

** (highest precedence)
*, /, %, //
+, -
<, <=, >, >=, !=, ==
not
and
or (lowest precedence)

Summary

Understanding Python operators is fundamental for writing expressions and performing various operations. They provide powerful tools for manipulating data and controlling program flow.
