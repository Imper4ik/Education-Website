Python Data Types

Introduction

Data types in Python define the type of data a variable can hold. Python is dynamically typed, meaning you don't need to declare the type of a variable explicitly. The interpreter determines the type dynamically based on the value assigned to it.

Numeric Types

Python supports several numeric types:

  • int - Integer numbers (e.g., 5, -10)
  • float - Floating-point numbers (e.g., 3.14, -2.5)
  • complex - Complex numbers (e.g., 2 + 3j)
num1 = 10
num2 = 3.14
num3 = 4 + 5j

Boolean Type

The bool type represents boolean values, True or False:

flag = True
is_valid = False

Sequence Types

Python has several sequence types:

  • str - String type for text (e.g., "Hello, Python")
  • list - Ordered collection of items (e.g., [1, 2, 3])
  • tuple - Immutable ordered collection (e.g., (1, 2, 3))
message = "Hello, Python"
my_list = [1, 2, 3]
my_tuple = (4, 5, 6)

Set Types

The set type represents unordered collections of unique items:

my_set = {1, 2, 3}

Mapping Type

The dict type is a collection of key-value pairs:

my_dict = {"name": "John", "age": 30}

None Type

The NoneType represents a null or absence of value:

my_var = None

Summary

Understanding Python's data types is fundamental for writing effective and structured code. Python's dynamic typing and rich set of built-in data types provide flexibility and power in handling various types of data and computations.
