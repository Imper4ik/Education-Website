Python Strings

Introduction

Strings in Python are sequences of characters, represented using single quotes (' ') or double quotes (" "). They support various operations and methods for manipulating and working with text data.

String Basics

Python provides many built-in functions and methods for working with strings:

len(), str(), format(), upper(), lower(), strip()

String Operations

Operations on strings include concatenation and repetition:

+, +=, *, *=

String Indexing and Slicing

Strings can be accessed using index numbers and sliced to extract substrings:

s = "Hello"
print(s[0])   # Output: 'H'
print(s[1:4]) # Output: 'ell'

String Methods

Python's string class provides a rich set of methods for string manipulation:

split(), join(), replace(), find(), count()

String Formatting

Strings can be formatted using f-strings or the format() method:

name = "Alice"
age = 30
print(f"My name is {name} and I am {age} years old.")
# Output: My name is Alice and I am 30 years old.

Unicode and Encoding

Python supports Unicode characters, allowing for internationalization and proper encoding:

text = "Καλημέρα"
print(text)  # Output: Καλημέρα

Summary

Understanding how to work with strings is essential for Python programming. With various operations, methods, and formatting options, Python strings provide powerful tools for text manipulation and data processing.
