Python String Operations

Introduction

Strings in Python are immutable sequences of characters. Python's `str` class provides a rich set of methods for string manipulation, including operations for concatenation, slicing, formatting, and more.

Basic String Operations

Here are some basic string operations provided by Python's `str`:

def main():
    # Example of string concatenation
    str1 = "Hello, "
    str2 = "world!"
    result = str1 + str2
    print("Concatenated string:", result)

    # Example of string length
    str_len = len("Example")
    print("Length of 'Example' is", str_len)

if __name__ == "__main__":
    main()

This example demonstrates string concatenation using `+` operator and calculating string length using `len()` function.

String Formatting

Formatting strings in Python:

def main():
    # Example of string formatting
    name = "Alice"
    age = 30
    formatted_str = f"My name is {name} and I am {age} years old."
    print(formatted_str)

if __name__ == "__main__":
    main()

This snippet uses formatted string literals (f-strings) for string interpolation.

String Methods

Commonly used methods for string manipulation:

def main():
    # Example of string methods
    text = "    Hello, world!    "
    print(text.strip())         # Removes leading and trailing whitespace
    print(text.lower())         # Converts string to lowercase
    print(text.replace("world", "Python"))  # Replaces substring
    print(text.startswith("Hello"))         # Checks if string starts with a substring

if __name__ == "__main__":
    main()

This code demonstrates methods like `strip()`, `lower()`, `replace()`, and `startswith()` for string manipulation.

Summary

Python's `str` class offers versatile methods for manipulating strings, making string handling in Python efficient and straightforward. Understanding these operations is essential for effective Python programming.
