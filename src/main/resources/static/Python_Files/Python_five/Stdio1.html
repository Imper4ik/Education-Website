Python Standard Input and Output

Introduction

In Python, standard input and output operations are handled through built-in functions for reading and writing data.

Basic Input and Output

Examples of basic input and output operations:

def main():
    # Example of standard output
    print("Hello, world!")

    # Example of standard input
    name = input("Enter your name: ")
    print("Hello,", name)

if __name__ == "__main__":
    main()

This example demonstrates using `print()` function for output and `input()` function for input.

File Input and Output

Reading from and writing to files in Python:

def main():
    # Example of file output
    with open('output.txt', 'w') as f:
        f.write("Hello, file!\n")

    # Example of file input
    with open('output.txt', 'r') as f:
        content = f.read()
        print("Content of 'output.txt':", content)

if __name__ == "__main__":
    main()

This snippet shows writing to a file using `open()` function with 'w' mode and reading from it using 'r' mode.

Formatted Output

Formatting output in Python:

def main():
    # Example of formatted output
    name = "Alice"
    age = 30
    print(f"My name is {name} and I am {age} years old.")

if __name__ == "__main__":
    main()

This code uses formatted string literals (f-strings) for string interpolation in output.

Summary

Python provides versatile functions and methods for input and output operations, enabling effective interaction with users and handling of data through files. Understanding these operations is fundamental for Python programming.
