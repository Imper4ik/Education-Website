Python Time Operations

Introduction

Python provides several modules for handling time-related operations, including retrieving current time, formatting dates, and performing date arithmetic.

Working with Dates and Times

Examples of basic date and time operations:

import datetime

def main():
    # Example of retrieving current date and time
    current_datetime = datetime.datetime.now()
    print("Current datetime:", current_datetime)

    # Formatting dates
    formatted_date = current_datetime.strftime("%Y-%m-%d %H:%M:%S")
    print("Formatted datetime:", formatted_date)

    # Date arithmetic
    future_date = current_datetime + datetime.timedelta(days=7)
    print("Future date:", future_date.strftime("%Y-%m-%d"))

if __name__ == "__main__":
    main()

This example demonstrates retrieving current date and time, formatting dates using `strftime()`, and performing date arithmetic with `timedelta`.

Time Zones

Working with time zones:

import pytz

def main():
    # Example of working with time zones
    eastern = pytz.timezone('US/Eastern')
    local_time = datetime.datetime.now(eastern)
    print("Local time (Eastern Time Zone):", local_time.strftime("%Y-%m-%d %H:%M:%S %Z%z"))

if __name__ == "__main__":
    main()

This snippet shows how to work with different time zones using the `pytz` module.

Summary

Python's datetime module and third-party libraries like `pytz` provide robust capabilities for handling date and time operations, making it versatile for applications requiring time-based calculations and formatting.
