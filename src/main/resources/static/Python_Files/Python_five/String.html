Python String Operations

Introduction

Strings in Python are sequences of characters, and Python provides many built-in functions and methods to manipulate and work with strings efficiently.

Basic String Operations

Examples of basic string operations:

def main():
    # Example of basic string operations
    message = "Hello, World!"
    print("Original:", message)

    # Length of a string
    print("Length:", len(message))

    # Accessing characters
    print("First character:", message[0])
    print("Last character:", message[-1])

    # Slicing a string
    print("Sliced message:", message[7:])

    # Concatenation
    name = "Alice"
    greeting = "Hello, " + name
    print("Concatenated:", greeting)

if __name__ == "__main__":
    main()

This example demonstrates operations such as accessing characters, slicing, and concatenation.

String Methods

Examples of using string methods:

def main():
    # Example of string methods
    message = "   Hello, World!   "
    print("Original message:", message)

    # Removing whitespace
    stripped_message = message.strip()
    print("Stripped:", stripped_message)

    # Upper and lower case
    print("Uppercase:", message.upper())
    print("Lowercase:", message.lower())

    # Finding substrings
    print("Index of 'World':", message.find("World"))

    # Replacing text
    new_message = message.replace("World", "Python")
    print("Replaced message:", new_message)

if __name__ == "__main__":
    main()

This snippet shows methods like `strip()`, `upper()`, `lower()`, `find()`, and `replace()` for string manipulation.

Formatted Strings

Creating formatted strings in Python:

def main():
    # Example of formatted strings
    name = "Alice"
    age = 30
    print(f"My name is {name} and I am {age} years old.")

if __name__ == "__main__":
    main()

Formatted string literals (f-strings) allow for easy interpolation of variables into strings.

Summary

Python provides powerful tools for working with strings, including basic operations, methods for manipulation, and formatted string literals. Understanding these features is essential for effective string handling in Python programming.
