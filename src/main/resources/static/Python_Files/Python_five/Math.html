Python Math Operations

Introduction

In Python, mathematical operations are performed using basic operators and functions provided by the `math` module for more advanced operations.

Basic Math Operations

Examples of basic arithmetic operations:

def main():
    # Example of basic arithmetic operations
    a = 10
    b = 5
    print(f"Addition: {a} + {b} = {a + b}")
    print(f"Subtraction: {a} - {b} = {a - b}")
    print(f"Multiplication: {a} * {b} = {a * b}")
    print(f"Division: {a} / {b} = {a / b}")

if __name__ == "__main__":
    main()

This example demonstrates addition, subtraction, multiplication, and division using basic Python operators.

The Math Module

Using functions from the `math` module:

import math

def main():
    # Example of using math module functions
    num = 16
    print(f"Square root of {num}: {math.sqrt(num)}")
    print(f"Factorial of {num}: {math.factorial(num)}")
    print(f"Power of 2^3: {math.pow(2, 3)}")

if __name__ == "__main__":
    main()

This snippet shows examples of using `math.sqrt()` for square root, `math.factorial()` for factorial, and `math.pow()` for exponentiation.

Summary

Python provides powerful built-in capabilities for performing basic and advanced mathematical operations. Utilizing operators and functions from the `math` module allows for efficient computation in Python programs.
