Python Classes

Introduction

Classes in Python are blueprints for creating objects. They define attributes (variables) and methods (functions) that characterize the objects. Classes provide structure and behavior to objects, enabling code reuse and maintaining clean code.

Defining a Class

A class is defined using the class keyword, followed by the class name and a colon. Inside the class body, you define attributes and methods.

class MyClass:
    def __init__(self, param1, param2):
        self.attribute1 = param1
        self.attribute2 = param2

    def method1(self):
        # Method definition
        pass

Creating Objects (Instances)

An object or instance of a class is created using the class name followed by parentheses (). This invokes the class's constructor method (__init__) to initialize the object.

# Creating an object of MyClass
obj = MyClass("value1", "value2")

Attributes and Methods

Attributes are variables bound to an instance or class, accessible using dot notation (obj.attribute1). Methods are functions defined within a class, accessed similarly (obj.method1()).

Constructor Method (__init__)

The __init__ method initializes new instances of a class. It is called automatically when an object is created, allowing initialization of attributes.

class MyClass:
    def __init__(self, param1, param2):
        self.attribute1 = param1
        self.attribute2 = param2

Class Inheritance

Python supports inheritance, allowing one class (subclass) to inherit attributes and methods from another (base or parent class). This promotes code reuse and hierarchy.

# Parent class
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError("Subclass must implement abstract method")

# Child class inheriting from Animal
class Dog(Animal):
    def speak(self):
        return "Woof!"

Encapsulation

Encapsulation is achieved by bundling data (attributes) and methods that operate on the data within a single unit (class). Access modifiers like public, protected, and private help control access to class members.

Summary

Classes in Python are fundamental for object-oriented programming, providing structure, encapsulation, and inheritance. Understanding how to define, instantiate, and use classes is essential for building robust and maintainable applications.
