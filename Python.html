Python

Python is an interpreted programming language with a strong and dynamic type system and automatic memory management. Thanks to its rich standard library, ease of writing and clear syntax, software developed in it is created very quickly. Hence, it is promoted as a language for prototyping applications. The general recommended process for developing software in Python is as follows:

Introduction

Syntax - Control statements - Operators - Data types - Functions - Namespaces - Arrays - Character strings

Object-oriented programming

Classes - Access control - Constructors - Destructors - Inheritance - Virtual functions - Abstract classes - Overloading operators

Syntax and language elements

Exception handling - Function templates - Class templates
