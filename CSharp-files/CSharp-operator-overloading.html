Operator Overloading in C#

Operator overloading allows you to define custom behavior for operators such as +, -, *, /, etc., for user-defined types. It provides a convenient way to extend the functionality of operators to work with your own types.

Here's an example demonstrating the usage of operator overloading:

        
using System;

class Complex
{
    public int Real { get; set; }
    public int Imaginary { get; set; }

    public Complex(int real, int imaginary)
    {
        Real = real;
        Imaginary = imaginary;
    }

    public static Complex operator +(Complex c1, Complex c2)
    {
        return new Complex(c1.Real + c2.Real, c1.Imaginary + c2.Imaginary);
    }

    public override string ToString()
    {
        return Real + " + " + Imaginary + "i";
    }
}

class Program
{
    static void Main()
    {
        Complex num1 = new Complex(3, 4);
        Complex num2 = new Complex(2, 5);

        Complex sum = num1 + num2;

        Console.WriteLine("Sum: " + sum); // Output: Sum: 5 + 9i
    }
}

In this example, the `Complex` class defines the addition operator `+` to work with complex numbers. This allows us to use the `+` operator to add two `Complex` objects, simplifying the code and making it more intuitive.
