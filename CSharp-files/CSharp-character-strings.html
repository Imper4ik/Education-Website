Character Strings in C#

In C#, a string is an object of the System.String class, which represents a sequence of characters. Strings are used extensively in programming to manipulate and store text.

Here are some common operations you can perform with strings in C#:

  • Creating and initializing strings
  • Concatenating strings
  • Accessing characters in a string
  • Comparing strings
  • Finding substrings
  • Replacing parts of a string

Here's an example demonstrating some of these operations:

        
using System;

class Program
{
    static void Main()
    {
    // Creating and initializing strings
    string greeting = "Hello, world!";
    string name = "Alice";

    // Concatenating strings
    string message = greeting + " My name is " + name + ".";
    Console.WriteLine(message);

    // Accessing characters in a string
    char firstChar = greeting[0];
    Console.WriteLine("The first character of the greeting is: " + firstChar);

    // Comparing strings
    string name2 = "Bob";
    bool areEqual = name == name2;
    Console.WriteLine("Are the names equal? " + areEqual);

    // Finding substrings
    bool containsWorld = greeting.Contains("world");
    Console.WriteLine("Does the greeting contain 'world'? " + containsWorld);

    // Replacing parts of a string
    string newGreeting = greeting.Replace("world", "everyone");
    Console.WriteLine(newGreeting);
}

In the above example, various string operations are demonstrated, such as creating and initializing strings, concatenating them, accessing individual characters, comparing strings, finding substrings, and replacing parts of a string.
